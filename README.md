## Repository Overview

This repository contains all the lab assignments completed as part of the Data Mining & Visualization coursework. Each assignment demonstrates different concepts of data handling, visualization, and basic animation using Python.

The tasks range from simple static plots to dynamic and animated visualizations, along with data preprocessing techniques like handling missing values and detecting outliers.
